# Extension Icons

This directory contains the extension icons in various sizes:

- `icon-16.png` - 16x16px (toolbar icon)
- `icon-32.png` - 32x32px (Windows)
- `icon-48.png` - 48x48px (extension management page)
- `icon-128.png` - 128x128px (Chrome Web Store)

## Creating Icons

To create the PNG icons from the SVG source:

1. Use the `icon.svg` as the source
2. Export at the required sizes
3. Ensure they maintain the gradient and visual clarity at small sizes
4. Use PNG format for better compatibility

## Icon Design

The icon features:
- Electric blue to purple gradient
- Lightning bolt symbolizing speed and optimization
- Dark background for contrast
- Sparkle effects for premium feel
- Clean, modern design that works at small sizes
